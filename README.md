# Patient–Doctor Connect

A React app (TanStack Start). View it in your browser at `http://localhost:8080` while the dev server runs.

## Setup

1. Install Node.js from https://nodejs.org (LTS).
2. Open Terminal in this folder.
3. Install dependencies (once):
   ```bash
   npm install
   ```
4. Start the app:
   ```bash
   npm run dev
   ```
5. Open `http://localhost:8080` in your browser.

Press `Ctrl + C` to stop.
